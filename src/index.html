  1. 记账吧

    记账吧是一款极简的记账应用，基于 Vue 和 TypeScript 实现，

  2. Figure Bed

    源码链接 项目预览

    Figure Bed 是使用 React 技术栈实现一个在线图床，

    核心功能包括：登录、注册、图片上传、图片大小定制、上传历史展示等。

    使用 React-Router 进行页面跳转，使用 LeanCloud 做为后端保存数据，

    此外用到的技术还有：react-create-app、styled-components、Mobx、Antd 等。

  3. Something Different 小程序

    源码链接

    Something Different 小程序是一款能给微信头像添加图标的小程序。

    添加的图标和被添加图标的图片都支持自定义，同时用户可以对图标进行移动、缩放和旋转的操作。

    操作完图标后通过 canvas 标签将图标和图片叠加起来，并保存至用户手机上。
